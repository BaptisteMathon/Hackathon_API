Fonctionnement de notre API pour le projet Hackathon:

Afin d'utiliser les routes suivantes, il faut obligatoirement être connecté:

Les GET:

  • '/allCars'Renvoie toutes les voitures disponible à la location
  • '/car/:id'Renvoie le détail d'une voiture, en y insérant son id
  • '/allUsers'Renvoie tous les utilisateurs inscrits
  • '/user/:id'Renvoie les informatons d'un utilisateur, en y insérant son id
  • '/allLocations'Renvoie toutes les locations de voitures, avec l'id du véhicule, la date, et l'id de l'utilisateur

Les POST:

  • '/createCars'Permet d'ajouter un nouveau véhicule à la location
  • '/createLocation'Permet d'ajouter une nouvelle location, lorsqu'un utilisateur souhaite loué un véhicule

Le PUT:

  • '/updateReservation/:id'Permet de modifier la date d'une location

Les DELETE:

  • '/deleteCars'Permet à un utilisateur de supprimer sa voiture de la location
  • '/deleteLocation'Permet à un utilisateur de supprimer une de ses locations
  • '/deleteUser'Permet à l'utilisateur de supprimer son compte
